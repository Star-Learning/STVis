# Storyboard - Episode 001: World GDP Shift

## Episode Overview

- **Title:** 世界经济的重心，正在向哪里移动？
- **English:** Where Is the Center of the World Economy Moving?
- **Duration:** about 70-80 seconds
- **Format:** 16:9 landscape, with optional 9:16 derivative

## Scene Breakdown

### Scene A: Cold Open (0-8s)

- Dark cinematic background
- World map outline fades in
- A few major economies begin to glow
- Title appears softly

### Scene B: Global Setup (8-20s)

- Start from an early year
- Countries fill by GDP share of world total
- Year label begins advancing
- Narration establishes that economic weight is not static

### Scene C: Decades in Motion (20-35s)

- Year-by-year transitions accelerate
- Ranking panel updates as leaders shift
- Viewer sees broad redistribution of global weight

### Scene D: Country Focus (35-50s)

- Hold briefly on several notable trajectories
- Keep emphasis clean and selective
- Use top-country labels and short captions

### Scene E: Center Motif (50-65s)

- Weighted economic-balance marker moves across the map
- Narration frames it as a visual proxy rather than a strict scientific claim

### Scene F: Closing (65-80s)

- Return to full world view
- End on the latest year
- Close with data + code open-source note
